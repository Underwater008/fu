# Fu Project

## Tech Stack
- Vite (v7.3.1) - build tool and dev server
- Three.js (v0.182.0) - 3D rendering

## Commands
- `npm run dev` / `npx vite --open` - Start dev server (opens browser automatically)
- `npm run build` - Build for production (outputs to `dist/`)
- `npm run preview` - Preview production build

## Notes
- This is a Vite project. Use `npm run dev` to open in browser, not `open index.html` directly.
